## About the API

An API to provide information about the Brazilian football teams that are playing in the A and B series of the Brazilian football championship.
## Technologies used

* **Java 11 (Java Development Kit - JDK: 11.0.12)**
* **Spring Boot 2.7.3**
* **Maven**
* **Model Mapper 3.1.0**
